# FashionMNIST 自监督对比学习项目 (SimCLR)

基于 **SimCLR (Simple Framework for Contrastive Learning of Visual Representations)** 算法，在不使用任何类别标签的情况下，对 FashionMNIST 数据集进行无监督特征学习。

通过精细化的数据增强与特征归一化技术，模型训练完成后，高维特征能够在降维空间中自然分离，呈现出完美的“**物理聚类海岛**”分布——各类别的衣服（如鞋子、包包、衣服）高度聚集成簇，且簇与簇之间边缘界限清晰。

---

## 📂 项目文件说明

- **`train_and_tsne.py`**：核心训练脚本。包含自监督对比学习训练循环、在线 KNN 评估以及训练结束后的 t-SNE 特征可视化。
- **`evaluate_accuracy.py`**：独立评估脚本。加载训练好的权重文件（`simclr_fashionmnist.pth`），运行高精度全量 KNN 评估与线性探针（Linear Probe）评估。
- **`plot_tsne_only.py`**：独立绘图与调参脚本。读取训练生成的特征数据（`tsne_data.npz`），可快速调整 t-SNE 画图参数和着色模式，无需重新训练。
- **`test_gpu.py`**：显卡加速测试脚本。

---

## 🧠 核心算法与计算方法

### 1. 图像增强设计 (Data Augmentations)
为了让自监督网络关注衣物的“全局几何轮廓”而非局部像素细节，本项目摒弃了标准的 ImageNet 暴力裁剪，采用了对 FashionMNIST 最优的增强组合：
- **3通道灰度映射**：将灰度图扩展为 3 通道，无缝对接 ResNet-18 骨干网络的表达能力。
- **随机仿射变换 (RandomAffine)**：允许旋转（25度内）、平移（上下20%/左右10%）、缩放（0.7至1.3倍）。这是特征完美解耦的关键。
- **水平翻转 (RandomHorizontalFlip)**：概率 0.5。
- **精准统计归一化**：采用 FashionMNIST 数据集真实的均值与方差：mean = (0.286, 0.286, 0.286), std = (0.353, 0.353, 0.353)。

### 2. 对比损失函数 (InfoNCE Loss)
对比学习的核心计算目标是：**拉近同一张图像不同增强版本之间的距离，推开不同图像之间的距离。**
- 投影特征表示为 z_i 与 z_j（来自于同一张图的两个增强样本）。
- 使用余弦相似度计算特征匹配度：
  sim(z_i, z_j) = (z_i · z_j) / (||z_i|| * ||z_j||)
- 通过温度系数 temperature = 0.1 缩放后，使用交叉熵计算 InfoNCE Loss。极低的温度系数（0.1）促使同类特征在高维空间中急剧收缩，形成超高内聚性。

### 3. 无监督特征质量评估
为了在**不污染训练模型、不泄漏标签**的前提下监测特征好坏：
- **在线 KNN 监测**：在每轮训练结束时，模型自动进入 eval 模式，冻结参数。对测试集特征和训练集特征跑一次 KNN (k = 20) 评估，得出无监督精度的客观数字。
- **线性探针 (Linear Probe)**：在 `evaluate_accuracy.py` 中，彻底冻结提取器，在其外部训练一个极简的单层线性层（分类器），用于测试特征的可分性上限。

## 🎨 t-SNE 完美分簇技巧与参数调整

最终的可视化图像能呈现出**“物理聚类海岛”**的效果，主要归功于以下技术参数的配合与调整：

### 1. 特征 L2 归一化 (L2 Normalization) —— 关键点
*   **计算方法**：
    h_norm = h / (||h||_2 + 1e-8)
    (即：将提取出的 512 维特征向量 h，除以该向量的二范数/长度，并加上一个极小值 1e-8 防止除以零)
*   **为什么有效**：自监督模型学习的是**特征的方向夹角（余弦相似度）**而非模长。在降维前对提取的特征进行 L2 归一化，能抹去向量长度对欧氏距离计算的干扰。这使得 PCA 降维和 t-SNE 能够百分之百反应模型真实的语义聚类情况。

### 2. t-SNE 降维超参数调整说明 (`plot_tsne_only.py`)
如果您需要微调绘图效果，请打开 `plot_tsne_only.py` 的 `plot_tsne` 函数，调整以下参数：

*   **`perplexity` (困惑度，默认值：50)**：
    - *物理含义*：估计每个数据点周围紧密邻居的数量。
    - *调整效果*：
        - 若设为 **`50 ~ 80`**：图表会更关注大尺度的全局拓扑结构，**各“海岛”内部会收缩得更圆润紧凑，类别边界更干净。**
        - 若设为 **`20 ~ 30`**：更关注微观局部，可能会把同一个大类细分成几个碎小的“岛礁”（例如把运动鞋和踝靴切成独立的小簇）。
*   **`early_exaggeration` (早期夸大系数，默认值：12.0)**：
    - *物理含义*：控制在优化早期，不同簇之间被弹开的排斥力大小。
    - *调整效果*：
        - 调大到 **`15.0 ~ 20.0`**：会强行拉大不同簇之间的**空白海洋间距**，使“物理聚类海岛”在图纸上分布得更开，大幅减少不同类别散点边缘的交织。
*   **`init` (初始化方式，默认值：'pca')**：
    - 采用稳定的 PCA 进行初始化，能保证高维空间的大局结构被带入低维，相比随机初始化，能够生成具有更清晰大尺度几何边界的分簇。

## 🚀 运行指南

### 1. 开始训练与自动绘图
在终端中执行：
```bash
python train_and_tsne.py
```
- 模型将利用 GPU 加速跑完 100 轮训练。
- 期间每轮会输出当前 Loss 和在线 KNN 预测准确率。
- 训练结束后，自动保存权重至 `simclr_fashionmnist.pth`。
- 自动提取测试集前 2000 个样本，进行 L2 归一化 + PCA + t-SNE 降维，并保存为 **`tsne_result.png`**。

### 2. 独立运行高精度评估
在训练完成后，运行：
```bash
python evaluate_accuracy.py
```
- 计算并输出全量 10,000 张测试集图片的 KNN 精度。
- 冻结提取器，训练 10 轮线性探针（Linear Probe），给出特征分类的真实上限。

### 3. 快速调整参数并重新绘图 (无需重新训练)
如果您想直接改变绘图风格或配色方式，直接运行：
```bash
python plot_tsne_only.py
```
您可以通过修改 `plot_tsne_only.py` 中的 `main()` 函数里的 `mode` 变量来切换三种画图模式以应对不同的客户演示场景：
1. **`mode = "kmeans"` (完全无监督聚类着色)**：最符合客户不体现真实标签的要求。通过 K-Means 自动对特征点聚为 10 类并涂色，同时输出 NMI 分数，向客户展现其天然的边界分离。
2. **`mode = "uncolored"` (单色几何分簇)**：所有点统一着色。纯粹向客户展示未被标签污染的高维特征在几何空间中自然形成的离散“孤岛”。
3. **`mode = "supervised"` (真实标签着色)**：用 FashionMNIST 真实的衣服类别染色，用于内部开发及效果验证。
